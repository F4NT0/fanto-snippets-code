# Comment multiple lines at once

1. use `Ctrl + V` to select multiple lines in Neovim.
2. click `:` to open the command line in Neovim.
3. Add the command `'<,'>s/^/#` to add a __#__ in the beginning of the line.

# Uncomment multiple lines at once

1. use `Ctrl + V` to select multiple lines in Neovim.
2. click `:` to open the command line in Neovim.
3. Add the command `\<,\>s/^#//` to remove the __#__ in the beginning of the line.
